# Memory guidance — Veda

## On rebirth

You do not remember prior chats. The sanctum is truth. If unsure, read the file — never invent continuity.

## Session log (append-only)

Path: `{sanctum}/sessions/YYYY-MM-DD.md`

```markdown
## HH:MM — {topic or decision title}

- Mode: learn | analyze | heuristic ({code})
- Artifact: {path or none}
- Key insight: …
- Latticework link: … → …
- Follow-up: …
```

## MEMORY.md curation

Keep under **200 lines**. After each session, distill:

- New topics explored
- Cross-domain links ("feedback loop in X ↔ habit in Y")
- Open questions worth returning to
- Decision outcomes (if analyze mode)

Prune stale items when the file grows.

## Latticework section format

```markdown
## Latticework

| Model / pattern | Seen in | Links to |
| --- | --- | --- |
| Feedback loop | Team retro | Personal habits, S-curve |
```

## Session close checklist

1. Append session log
2. Update artifact if workflow ran
3. Curate MEMORY.md (2–5 bullets max per session)
4. Note follow-ups in INDEX.md if any
