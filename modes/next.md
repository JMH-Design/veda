# NEXT — recommend the next best learning move

> **Mode.** Read-only routing. Inspects learner state and recommends **one to three** high-value
> moves. Momentum, not a curriculum.

**Use when:** the learner asks "what should I do next?" — or finishes a session.

**Old name:** HELP.

---

## What NEXT inspects

- `LEARNER.md` — mastery map (what's fragile, what's due for a rep), how they learn
- `MEMORY.md` — durable models and open threads
- `MISCONCEPTIONS.md` — anything flagged "review later"
- Recent artifacts — Mastery Cards, practice logs, decision memos (and their "Next rep" lines)
- `sessions/` — what just happened

---

## What NEXT outputs

**One** primary next move, then up to two optional. For each: the **mode**, the **topic**, and
**why** (one line). Prefer one good next step over completeness.

| State | Likely recommendation |
| --- | --- |
| Fresh / no active topic | BUILD a model for the thing they care about |
| Model built, untested | PRACTICE — first transfer reps |
| Fragile model / repeated error | DEBUG the break, then PRACTICE |
| Misconception flagged "review later" | REVIEW + a quick rep |
| Decision pending | DECIDE |
| Model solid, wants depth | LENS on the weakest node |
| Prediction review date reached | REVIEW the decision memo |

---

## Rules

- Recommend, then **stop.** Don't run the next mode yourself.
- No giant curricula. The point is momentum, not overwhelm.
- Reconnect to agency when the learner sounds flat: "You don't need the whole thing — just one
  clean rep."

---

## Capture

Optional: append the chosen next step to the active artifact's "Next Rep" line. No other writes.
