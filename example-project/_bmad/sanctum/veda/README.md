# Personal sanctum — run init-sanctum.py to populate

After running:

```bash
python3 skills/veda-agent/scripts/init-sanctum.py --sanctum ./_bmad/sanctum/veda --owner "Your Name"
```

This folder will contain:

- `INDEX.md`, `PERSONA.md`, `CREED.md`, `BOND.md`, `MEMORY.md`, `CAPABILITIES.md`
- `sessions/` — session logs

Do not commit personal sanctum files to public repos.
